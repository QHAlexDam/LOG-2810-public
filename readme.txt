Éléments importants pour utiliser le logiciel:

-Le logiciel demande le nom du fichier .txt pour load l'inventaire.
il faut entrer le nom avec l'extention. 
	ex. inventaire.txt
NB: le chemin vers le fichier txt est déja préconfiguré de façon relative 
et il faut absolument que le fichier txt soit dans le dossier src.
